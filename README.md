# Jeffpardy

A real-time trivia game built with ASP.NET Core, SignalR, React, and TypeScript.

## Quick Start

See **[DEVELOPMENT.md](DEVELOPMENT.md)** for full setup instructions, including GitHub Codespaces, local development, testing, and debug mode.

The fastest way to get started:

1. Open a **GitHub Codespace** on this repo (everything is pre-configured).
2. Run `npm run build && cd src/backend && dotnet run`.
3. Use `?debugMode=2` to skip the Azure Blob connection and use local test categories.
